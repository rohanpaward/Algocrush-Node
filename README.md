# Backend Project

This project is built with Node.js using a modular structure.  
All feature-related code resides in the `modules` folder, and database schemas are stored in the `schema` folder.  

---

## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone <repo-url>
cd <repo-name>

2. Install Dependencies
npm install

3. Run the Server
npm install nodemon
npx nodemon server.js

📂 Project Structure
├── modules/        # All feature-based modules
├── schema/         # Database schemas
├── server.js       # Entry point of the app
├── package.json    # Project dependencies & scripts
└── README.md       # Documentation
